# Medha Harini — Personal Portfolio

A personal portfolio website built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no libraries, no build tools.

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

---

## Folder Structure

```
portfolio/

├── css/
│   ├── style.css
│   ├── navbar.css
│   ├── hero.css
│   ├── about.css
│   ├── projects.css
│   ├── contact.css
│   └── responsive.css
├── images/
└── README.md
```

---

## Features

- Custom animated cursor
- Smooth scroll with active nav highlight
- Scroll reveal animations
- Animated skill bars
- Polaroid photo card with 3D tilt on hover
- Mobile responsive with hamburger menu
- Contact form with success message
- Back to top button

---

## How to Run

Open `index.html` directly in your browser.

Or use **Live Server** in VS Code — right click `index.html` and click Open with Live Server.

---

## How to Deploy

Push to GitHub and enable GitHub Pages under Settings → Pages → main branch.

Or drag the project folder into netlify.com for instant deployment.
